# Smile Hub Frontend Direction

## Visual system

- Warm premium clinic palette with ivory base, charcoal ink, muted gold accents, and soft sage highlights
- Editorial display typography paired with refined sans body copy
- Layered cards, rounded surfaces, and subtle shadows instead of flat template sections

## Mobile rules

- Sticky thumb-zone CTA dock
- Swipeable reviews and treatment cards
- Short sections with generous spacing
- Light parallax/reveal accents only in key hero moments

## Key page rhythms

- Home: hero, proof, services, trust, reviews, gallery, team, contact
- Treatment pages: hero, benefits, process, FAQs
- Book: WhatsApp shortcuts, form fallback, call option
- Contact: map, hours, direct actions
